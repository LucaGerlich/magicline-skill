# Get utilization for the studio.

Return information about the utilization of this studio. Needs to be enabled for public view in the configuration.

Endpoint: GET /connect/v1/studio/{studioId}/utilization
Version: 1.0.0

## Path parameters:

  - `studioId` (integer, required)

## Response 200 fields (*/*):

  - `endTime` (object)

  - `endTime.hour` (integer)

  - `endTime.minute` (integer)

  - `endTime.nano` (integer)

  - `endTime.second` (integer)

  - `items` (array, required)

  - `items.isCurrent` (boolean, required)

  - `items.level` (string, required)
    Enum: "HIGH", "NORMAL", "LOW"

  - `items.percentage` (integer, required)

  - `items.startTime` (object, required)
