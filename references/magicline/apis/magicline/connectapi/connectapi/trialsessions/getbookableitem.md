# Get free slots for a trial session

Returns free slots for a studio where a trial session appointment can be booked.

Endpoint: GET /connect/v1/trialsession
Version: 1.0.0

## Query parameters:

  - `studioId` (integer, required)
    Example: 1

  - `startDate` (string, required)
    Example: "2021-05-15"

  - `endDate` (string, required)
    Example: "2021-05-21"

## Response 200 fields (*/*):

  - `bookingWithoutResourcesAllowed` (boolean)
    true: Customer can decide if a trainer is needed, false: Trainer is always mandatory (trainerRequired)

  - `description` (string)
    Short description how the trial session looks like

  - `name` (string)
    Display name of trial session

  - `slots` (array)

  - `slots.endDateTime` (string)

  - `slots.startDateTime` (string)
