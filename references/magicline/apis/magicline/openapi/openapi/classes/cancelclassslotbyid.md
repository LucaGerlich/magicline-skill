# Cancel a class booking

Required Scopes: 

Cancel booking of a class slot for given bookingId

Endpoint: DELETE /v1/classes/booking/{bookingId}
Version: 1.11.0
Security: ApiKeyAuth

## Path parameters:

  - `bookingId` (integer, required)
    Unique id of the booking to be cancelled
