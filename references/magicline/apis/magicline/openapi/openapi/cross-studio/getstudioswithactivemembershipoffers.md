# Get studios with active membership offers

Required Scopes: 

Returns all studios with at least one active membership offer.

Endpoint: GET /v1/cross-studio/membership-offers/studios
Version: 1.11.0
Security: ApiKeyAuth

## Response 200 fields (application/json):

  - `name` (string)
    Studio name
    Example: "Example studio"

  - `description` (string)
    Studio description
    Example: "Example studio description"

  - `country` (string)
    Studio address' country
    Example: "Germany"

  - `countryCode` (string)
    Studio address' country code
    Example: "DE"

  - `studioId` (integer, required)
    Studio Id
    Example: 123

  - `openingDate` (string)
    Studio opening date
    Example: "2025-06-11"

  - `closingDate` (string)
    Studio closing date
    Example: "2029-01-01"
