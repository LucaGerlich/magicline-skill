# withdraw cancelation of the additional module contract

Required Scopes: 

Withdraws additional module contract cancelation amendment or reverts cancelation for specific additional module contract id

Endpoint: POST /v1/memberships/{contractId}/self-service/additional-module-contracts/{additionalModuleContractId}/withdraw-cancelation
Version: 1.11.0
Security: ApiKeyAuth

## Path parameters:

  - `contractId` (integer, required)
    Unique ID of the contract

  - `additionalModuleContractId` (integer, required)
    Unique ID of the additional module contract

## Response 200 fields (application/json):

  - `success` (string)
