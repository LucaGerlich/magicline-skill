# withdraw cancelation of contract

Required Scopes: 

Withdraws contract cancelation amendment or reverts cancelation for specific contract id

Endpoint: POST /v1/memberships/{customerId}/self-service/withdraw-ordinary-contract-cancelation/{contractId}
Version: 1.11.0
Security: ApiKeyAuth

## Path parameters:

  - `customerId` (integer, required)
    Unique ID of the customer

  - `contractId` (integer, required)
    Unique ID of the contract

## Response 200 fields (application/json):

  - `success` (string)
