# Get an additional module contract

Required Scopes: 

Get an additional module contract via additional module contract id

Endpoint: GET /v1/memberships/{contractId}/self-service/additional-module-contracts/{additionalModuleContractId}
Version: 1.11.0
Security: ApiKeyAuth

## Path parameters:

  - `contractId` (integer, required)
    Unique ID of the main contract

  - `additionalModuleContractId` (integer, required)
    Unique ID of the additional module contract

## Response 200 fields (application/json):

  - `name` (string, required)
    Name of the additional module contract
    Example: "drink flat"

  - `price` (object, required)
    Represents a financial data

  - `price.amount` (number, required)
    Amount of the finance data tuple
    Example: 20

  - `price.currency` (string, required)
    Currency of the finance data tuple
    Example: "EUR"

  - `startDate` (string, required)
    Start date of the additional module contract
    Example: "2022-01-15"

  - `endDate` (string)
    End date of the additional module contract
    Example: "2022-01-15"

  - `cancelationDate` (string)
    Cancelation date of the additional module contract
    Example: "2022-01-15"

  - `lastPossibleCancelationDate` (string)
    Last possible cancelation date of the additional module contract
    Example: "2022-01-15"

  - `cancelationReason` (string)
    Represents the reason of a contract cancelation
    Example: "Illness"

  - `contractCancelationStatus` (string)
    Cancelation status of the contract cancelation
    Enum: "CANCELED", "PENDING_VERIFICATION", "PENDING_WITHDRAWAL_VERIFICATION"

  - `contractCancelationCanBeWithdrawn` (boolean, required)
    Whether a contract cancelation for the additional module contract can be withdrawn
    Example: true

  - `trialPeriod` (object)
    Represents the trial period of an additional module

  - `trialPeriod.term` (object)
    Represents a term

  - `trialPeriod.term.value` (integer, required)
    The value of the term
    Example: 2

  - `trialPeriod.term.unit` (string, required)
    Represents a temporal unit
    Enum: "DAY", "WEEK", "MONTH", "YEAR"

  - `trialPeriod.startDate` (string)
    Trial period start date of an additional module contract
    Example: "2022-01-15"

  - `rateCodes` (array)
    Rate codes of this contract

  - `rateCodes.name` (string)
    The name of the rate code
    Example: "Standard Rate"

  - `rateCodes.identifier` (string)
    Unique identifier for the rate code.
    Example: "RC12345"

  - `availableCancelationDates` (array)
