# Create a lead customer

Required Scopes: 

Create a lead customer

Endpoint: POST /v1/trial-offers/lead/create
Version: 1.11.0
Security: ApiKeyAuth

## Request fields (application/json):

  - `leadCustomerData` (object, required)
    Full Lead customer data with address

  - `leadCustomerData.thirdPartyId` (string)
    Unique ID of the third party customer in the third party system
    Example: "A1000"

  - `leadCustomerData.firstname` (string, required)
    First name of the lead customer
    Example: "Max"

  - `leadCustomerData.secondFirstname` (string)
    Second first name of the lead customer
    Example: "Peter"

  - `leadCustomerData.lastname` (string, required)
    Last name of the lead customer
    Example: "Mustermann"

  - `leadCustomerData.secondLastname` (string)
    Second last name of the lead customer
    Example: "Meier"

  - `leadCustomerData.email` (string, required)
    Email of the lead customer
    Example: "example@email.com"

  - `leadCustomerData.gender` (string)
    Gender of the customer
    Enum: "MALE", "FEMALE", "UNISEX"

  - `leadCustomerData.dateOfBirth` (string)
    Date of birth of the lead customer
    Example: "2000-10-10"

  - `leadCustomerData.telephone` (string)
    Telephone number of the lead customer
    Example: "5006001112"

  - `leadCustomerData.language` (object)
    Language of the lead customer

  - `leadCustomerData.language.languageCode` (string, required)
    The language code
    Example: "de"

  - `leadCustomerData.language.countryCode` (string)
    The country code
    Example: "DE"

  - `leadCustomerData.address` (object)
    Lead customer address information

  - `leadCustomerData.address.street` (string, required)
    Street of the lead customer
    Example: "Burgring"

  - `leadCustomerData.address.houseNumber` (string)
    House number of the lead customer
    Example: "23A"

  - `leadCustomerData.address.zipCode` (string, required)
    Zip code of the lead customer
    Example: "13000"

  - `leadCustomerData.address.city` (string, required)
    City name of the lead customer
    Example: "Berlin"

  - `trialOfferConfigId` (integer, required)
    The ID of the trial offer config
    Example: 1

## Response 200 fields (application/json):

  - `leadCustomerId` (integer)
    The ID of the created lead customer
    Example: 1001
