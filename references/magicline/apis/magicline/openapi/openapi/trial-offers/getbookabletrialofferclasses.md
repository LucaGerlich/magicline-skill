# Get bookable trial offer classes

Required Scopes: 

Get pageable bookable trial offer classes within the next year

Endpoint: GET /v1/trial-offers/bookable-trial-offers/classes
Version: 1.11.0
Security: ApiKeyAuth

## Query parameters:

  - `offset` (string)
    Offset from last request (last ID of the class)

  - `sliceSize` (integer)
    Desired size of data chunk

## Response 200 fields (application/json):

  - `result` (array, required)
    List of trial offer classes

  - `result.trialOfferConfigId` (integer, required)
    Unique ID of the trial offer class
    Example: 1

  - `result.trialOfferClass` (object, required)
    Trial offer class

  - `result.trialOfferClass.id` (integer, required)
    Unique ID of the class
    Example: 1001

  - `result.trialOfferClass.title` (string)
    Title of the class
    Example: "Mission Beach body"

  - `result.trialOfferClass.type` (string)
    Defines whether a class is held in the studio or virtually
    Enum: "STUDIO", "STREAM"

  - `result.trialOfferClass.duration` (integer)
    Duration of the class in minutes
    Example: 45

  - `result.trialOfferClass.category` (string)
    Category of the class
    Example: "Fitness"

  - `result.trialOfferClass.description` (string)
    Description of the class
    Example: "Get your body ready for beaches"

  - `result.trialOfferClass.imgUrl` (string)
    Image URL assigned to display this class

  - `result.trialOfferClass.bookable` (boolean, required)
    Whether this class can be booked

  - `result.trialOfferClass.cancelCondition` (object)
    Cancelation condition of a class

  - `result.trialOfferClass.cancelCondition.cancelationAfterLatestPossible` (boolean)
    Whether cancelation after latest term is possible
    Example: true

  - `result.trialOfferClass.cancelCondition.latestCancelTerm` (object)
    The latest term before which the class can be canceled

  - `result.trialOfferClass.cancelCondition.latestCancelTerm.value` (integer, required)
    The value of the term with minutes
    Example: 2

  - `result.trialOfferClass.cancelCondition.latestCancelTerm.unit` (string, required)
    Represents a temporal unit with minutes
    Enum: "MINUTES", "HOURS", "DAY", "WEEK", "MONTH", "YEAR"

  - `result.id` (integer)

  - `hasNext` (boolean, required)
    True if there exists next data slice
    Example: true

  - `offset` (string, required)
    Offset for next query (last ID of the class)
    Example: "1234567890"
